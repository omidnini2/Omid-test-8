# راهنمای بخش فرانت‌اند (frontend)

این بخش رابط کاربری سایت را شامل می‌شود:
- تم لایت/دارک
- آپلود و ضبط صدا
- وارد کردن متن (تا ۱۰۰,۰۰۰ کاراکتر)
- دانلود و اشتراک‌گذاری خروجی
- نمایش زبان‌های پشتیبانی‌شده

## نحوه اجرا

کافی است پوشه frontend را روی هر هاست یا لوکال (WAMP/XAMPP/KS Web) قرار دهید و فایل index.html را باز کنید.

ارتباط با هوش مصنوعی از طریق API برقرار می‌شود (در پوشه api توضیح داده شده است).
