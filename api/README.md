# راهنمای بخش API

این بخش واسط بین فرانت‌اند (سایت) و بک‌اند (هوش مصنوعی) است.

## قابلیت‌ها
- دریافت فایل صوتی و متن از کاربر
- ارسال به سرور هوش مصنوعی (backend)
- دریافت خروجی صوتی و ارسال به فرانت‌اند

## نحوه اجرا

### PHP (برای WAMP/XAMPP)
کافی است فایل‌های این پوشه را در هاست PHP خود قرار دهید.

### Node.js (برای سرورهای پیشرفته‌تر)
دستورالعمل راه‌اندازی در همین پوشه قرار می‌گیرد.

## نکته

آدرس سرور هوش مصنوعی (backend) باید در فایل کانفیگ مشخص شود.
